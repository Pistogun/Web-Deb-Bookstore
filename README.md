------------------------------
📌 PROJECT OVERVIEW
------------------------------
This project is a fully functional front-end E-Commerce Bookstore website built using:
- HTML5
- CSS3
- Bootstrap 5
- JavaScript (jQuery only)

It allows customers to:
✔ Browse books  
✔ Add books to a cart  
✔ Checkout via a modal form  
✔ Store orders in a local history system  

------------------------------
📁 FILE STRUCTURE
------------------------------
All files are placed in **a single folder** for simplicity.

1. `index.html` — Home page with featured books  
2. `products.html` — Full product listing  
3. `cart.html` — Cart page with checkout modal  
4. `history.html` — Shows all previous orders  
5. `script.js` — jQuery-only cart, checkout, and order history logic  
6. `style.css` — Basic styling for layout and cards  
7. `README.txt` — This documentation file  
8. `book1.jpg`, `book2.jpg`, ... — Sample images (you may use your own book covers)

------------------------------
🧾 FEATURES
------------------------------
✅ Add to Cart system (with jQuery)  
✅ Dynamic total price updates  
✅ Bootstrap modal checkout  
✅ Stores customer info and order details  
✅ Displays order history with:
   - Customer name
   - Email
   - Date & time
   - Total price
   - Quantity of items

------------------------------
🔧 HOW TO USE
------------------------------
1. Open `index.html` in your browser to start browsing.
2. Click "Add to Cart" on any book.
3. Go to the "Cart" page.
4. Click "✅ Checkout" to open the modal.
5. Fill out your name, email, and address.
6. Click "Place Order".
7. Visit the "Order History" page to view all past orders.

📝 NOTE:
All data (cart + orders) is stored using `localStorage`. It resets if the browser cache is cleared.
